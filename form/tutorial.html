About the Gun Deaths Editor Interface

Search for Victims

Fill out a field, and you'll see all victims that match it.

  • If you put "Jon Smith" in name, you'll see all victims with the exact name "Jon Smith"
  • If you put "Jon Smith" in name and "13" in age, you'll see all victims with the exact name "Jon Smith" who are 13

Learn about Quick Search below.

Add a Victim

On "Add," you can add a new victim.

As you start filling out the form, you'll start seeing existing victims with the same date and city to help you determine if the victim already exists.

After you've added a victim to the database, a sample tweet will appear.

Edit / Delete Victims

On either tab, whenever you see search results, you can edit them.

To delete: Press .

To edit: Press to unlock a row. Editable fields will appear blue. Press to save. The entry is not saved until the edit icon appears again.

Copy Victim

To copy some data from an existing victim, press . This will transfer you to Add Mode, with location, date, and URL already filled out.

Change History

Every time somebody makes a change to a victim, the property changed, its old value, its new value, the user who changed it, and the time it was changed is logged. To view the change history of a victim, press .

Sorting

Click a column to sort by it. Click it again to sort in the opposite direction.

Currently, you can only sort by one column at a time.

On "search", the default sort is by the time each victim was last modified. To restore this default sort, click the "ordering by [property]" at the top.

Neat Tricks

Smart Fields

When you fill out a field, the application will automatically interpret your input and fill out related fields.

Examples:

  • If you put "10" into age, ageGroup will automatically show "child"
  • If you put "adult" into age, ageGroup will show "adult"
  • If you put "tuesday" into date, date will show the date of last Tuesday
  • If you put "today" or "yesterday" into date, date will show the appropriate date
  • When you fill out city and state, the latitude and longitudes will automatically appear
  • "w", "f", and "m" in gender will automatically correct to either "M" or "F"

If the application doesn't understand your input, the field will remain blank.

Quick Search

Use "Quick Search" to rapidly search for keywords or multiple fields at once. For example:

  • "teen f": find all teenage female victims
  • "1230": find victim with victimID 1230.
  • "m Chicago": find all 21-year-old male victims who have "Chicago" in any property
  • "Smith 1/2/2013": find all Jan. 2 victims who have "Smith" in any property, even if it's only part of that property as in "Jon Smith"

The order in which terms are entered does not matter.
